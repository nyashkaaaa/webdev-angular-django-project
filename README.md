## 📌 Project Description
Anime Tracker is a web application where users can browse anime, add them to their personal list, track watching status, rate anime, and leave reviews.

## ⚙️ Technologies
- Angular
- Django
- Django REST Framework
- PostgreSQL

## 🚀 Features
- User registration & login (JWT)
- Browse anime list
- Add anime to personal list
- Update status (Watching, Completed, etc.)
- Rate anime
- Leave reviews
